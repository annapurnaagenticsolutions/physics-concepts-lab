# Factual Source Ledger

This lab uses factual content intended for Class 1–10 concept learning. The primary reference baseline for India-facing school alignment is:

1. NCERT official textbook portal: https://www.ncert.nic.in/textbook.php
2. CBSE Academic curriculum/syllabus portal: https://cbseacademic.nic.in/

Use this file as a release-gate ledger. Each future new lesson should map its key claims to one of the reference areas below before the lab is called fact-verified.

## Verification categories

- **Core concept accuracy:** formula, rule, process, definition, diagram logic.
- **Age fit:** whether the wording is appropriate for the intended class band.
- **Safety:** whether home/classroom activities avoid dangerous demonstrations.
- **Caveat quality:** whether simplified statements avoid becoming false.
- **Assessment reliability:** whether Try/Mini Test answers accept correct reasoning, not just one fragile word.

## Release stance

This package is QA-polished and source-ledger ready. It should not be marketed as a fully curriculum-certified textbook replacement. It is a static learning aid that should be used with teacher/parent review where needed.
