# Physics Concepts Lab — Class 1 to 10

Static prototype for the STEM Lab initiative.

## Scope

This build focuses only on **Physics** from Class 1 to Class 10.

- Class 1–5: observation-first physics foundations
- Class 6–8: structured concepts with units, diagrams, and rules
- Class 9–10: formal equations, graphs, ray diagrams, circuits, and numerical solving

## Learning Flow

Choose Class → Choose Chapter → Observe → Concept → Visual → Equation/Rule → Why → Solve → Try → Mini Test

## Special Physics Visual Systems Included

- Force arrows
- Motion and graph visuals
- Light and ray diagrams
- Shadow formation
- Circuit diagrams
- Heat transfer maps
- Energy flow maps
- Magnetism visuals
- Pressure/friction visuals
- Gravity, work, power, and Ohm’s law interactions

## Design Principle

Physics and Chemistry should remain separate because Physics needs arrows, rays, graphs, circuits, fields, and motion paths, while Chemistry needs particles, atoms, molecules, reactions, pH scales, and balancing helpers.

## Usage

Open `index.html` in any modern browser. No server or external dependency is required.
